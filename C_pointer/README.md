# C_pointer
--------------

##指针的声明
C语言声明格式:"类型 变量名;"
- 基本类型:`int hoge;`
- 指针类型:`int *pointer;`

区别在于:

声明|含义|
---|--|--
int hoge;|声明**整数类型**的变量 hoge|
int *pointer;|声明 **"指向int的指针"**类型的变量 pointer| 

关于`int *pointer;` 其:

   |   | 
---|---|---
**指针类型**|指向int的指针  |
**指针类型的变量**|pointer  |
**指针类型的值**| 内存的地址 |
